# Snowflake Designer Lab - Cultural Winter Patterns

An interactive web application for creating symmetrical snowflake designs inspired by winter celebration patterns from cultures around the world.

**Live Demo:** [https://user319183.github.io/holiday-traditions/](https://user319183.github.io/holiday-traditions/)

## About

This project is a submission for the Winter Holiday & Traditions Web Design Challenge. Users can:

- Design snowflakes using brush tools with automatic symmetry (4-fold, 6-fold, 8-fold)
- Apply cultural pattern stamps from 12 different global traditions
- Save designs to a personal gallery stored in LocalStorage
- Export creations as PNG images
- Learn about the cultural significance of each pattern

## Cultural Patterns Featured

Patterns from 12 global winter traditions including:
- Indian (Diwali) - Rangoli circles
- Scandinavian (Yule) - Nordic snowflakes
- Jewish (Hanukkah) - Star of David
- Chinese (Lunar New Year) - Fortune coins
- Japanese (Shogatsu) - Cherry blossoms
- Mexican (Las Posadas) - Poinsettia flowers
- German (Weihnachten) - Traditional stars
- Russian (Novy God) - Matryoshka patterns
- Celtic (Winter Solstice) - Triquetra knots
- Persian (Yalda) - Pomegranate motifs
- African (Kwanzaa) - Unity symbols
- Native American - Four directions

## Technologies Used

- React with TypeScript
- Vue.js 3 (Pattern Library widget)
- HTML5 Canvas (Drawing and symmetry)
- Tailwind CSS
- LocalStorage API
- Vite

## Features

### Symmetry Drawing
Real-time symmetry reflection with 4-fold, 6-fold, or 8-fold options for creating snowflake patterns automatically.

### LocalStorage Persistence
- Snowflake gallery with thumbnails
- Dark/light mode preference
- Achievement tracking

### Responsive Design
- Mobile-friendly touch controls
- Glassmorphism UI
- Animated snowfall background
- Smooth transitions
